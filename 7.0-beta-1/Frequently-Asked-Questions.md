# Frequently Asked Questions

### What platforms does this engine support?

- Windows
- mac OS
- Android
- HTML5

This is partly due to our dependency on public libraries

### Which GameMaker version should I use with this engine?

We recommend GameMaker 2024.14+

### What is the difference between underengine and other existing engines?

- This engine has thorough documentation and comments to explain the inner workings
- This engine offers accessibility features that other engines have failed to deliver
- This engine offers support for MULTIPLE resolutions, giving players more options!

### I encountered a bug

File a bug report, either in the [Github Issues tab](https://github.com/neon-gr33n/underengine/issues) (Please label it as a "bug")
Please check if the issue has already been resolved before filing it.

### There is a feature that isn't in the engine, and I'd like to see it added!

Like with bug reports, you can file it in the [Github Issues tab](https://github.com/neon-gr33n/underengine/issues) (Please label it as a "enhancement")
Please check if this feature has been requested and/or denied prior to filing.

### Are there any existing projects that use underengine?

So far, only  [SWAPPED MYTH]([SWAPPED MYTH](https://swappedmyth.com/), but we're certain more projects will use this over time!
