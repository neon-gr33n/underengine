
## `loc_default_sprites()
` → `undefined`
Initializes and loads language-specific sprites based on current language setting

Cleans up existing sprites before loading new ones for the current language


## `loc_load_sprite(sprite_name, subimages, xorig, yorig)
` → *Number*
Loads a language-specific sprite from the appropriate folder

Searches in current language folder first, falls back to English if not found


| Parameter | Datatype  | Purpose |
|-----------|-----------|---------|
|`sprite_name` |String |Name of the sprite file (without extension)
 |
|`subimages` |Number |Number of subimages in the sprite
 |
|`xorig` |Number |X origin point for the sprite
 |
|`yorig` |Number |Y origin point for the sprite
 |

**Returns:** Sprite index if loaded successfully, -1 if sprite not found


## `loc_get_sprite(sprite_name)
` → *Number*
Retrieves a previously loaded language-specific sprite by name

Includes safety checks to ensure sprite still exists in memory


| Parameter | Datatype  | Purpose |
|-----------|-----------|---------|
|`sprite_name` |String |Name of the sprite to retrieve (as stored in loc_sprites)
 |

**Returns:** Sprite index if found and valid, -1 if sprite not found or deleted
