# Initialization

## `game_init()` → `undefined`
Initializes all core game systems and starts the game

## `initialize_audio_system()` → `undefined`
Sets up audio channels, buses, emitters, and default volumes

## `initialize_localization_system()` → `undefined`
Sets up language localization and font system

## `initialize_networking()` → `undefined`
Initializes network connections and GameJolt API

## `setup_game_constants()` → `undefined`
Sets core game constants and system configurations

## `setup_global_variables()` → `undefined`
Initializes global game state variables and settings

## `initialize_data_systems()` → `undefined`
Initializes game data systems (enemies, party, equipment, etc.)

## `setup_resolutions()` → `undefined`
Defines available screen resolutions for the game

## `setup_display()` → `undefined`
Configures display settings, GUI, and camera system

## `handle_error_recovery()` → `undefined`
Sets up crash handlers and handles recovery from previous crashes

## `setup_crash_handler()` → `undefined`
Creates exception handler for unhandled game crashes

## `handle_crash_file()` → `undefined`
Processes crash data from previous game crash

## `finalize_setup()` → `undefined`
Final configuration loading and setup before game start
